# Nigerian Regional Supply Chain Performance Dashboard — H1 2025

## Project Overview
This project analyzes supply chain performance across 6 Nigerian regions 
— Lagos, Abuja, Kano, Port Harcourt, Ibadan and Enugu — over 6 months 
from January to June 2025.

## Problem Statement
Management needed visibility into regional supply chain performance to 
identify delivery inefficiencies and their impact on returns and costs 
across all Nigerian regions.

## Tools Used
- Microsoft Power BI — Dashboard building and visualization
- Microsoft Excel — Data storage and preparation
- MySQL — Data querying and KPI calculation
- DAX — Calculated measures and color coding in Power BI

## KPIs Analyzed
- Order Fulfillment Rate
- On Time Delivery Rate (OTD)
- Return Rate
- Cost Per Order

## Key Findings
- Order volumes grew 39.3% from January to June across all regions
- Lagos is the highest performing region by order volume
- OTD rate consistently at 92% across all regions — below 95% benchmark
- Return rate consistently above 2% benchmark across all regions
- Cost per order stable at ₦2,000 across all regions
- OTD problem is the root cause driving excess returns and elevated costs

## Recommendations
- Conduct root cause analysis on consistently low OTD rate
- Review and strengthen logistics partner contracts and SLAs
- Investigate connection between late deliveries and rising returns
- Consider route optimization technology for Nigerian last mile delivery
- Fixing OTD performance is the single highest impact intervention available
